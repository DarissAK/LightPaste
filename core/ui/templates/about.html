About Light Paste

Light Paste is a lightweight paste service written in PHP using the Fat-Free Framework. It aims to provide a simple, customizable user interface and fast load times.

Keyboard Shortcuts

Light Paste supports keyboard shortcuts for several operations:

CTRL-ALT-S
Save paste
CTRL-ALT-Q
Toggle Line Numbers
CTRL-ALT-W
Toggle Word Wrap
CTRL-ALT-E
Toggle Smart Indent
CTRL-ALT-A
Toggle Bracket Matching
CTRL-ALT-Z
Toggle Tag Matching
CTRL-ALT-X
Toggle Active Line Highlighting
CTRL-ALT-C
Toggle Occurrence Highlighting
CTRL-ALT-V
Toggle Vertical Ruler
CTRL-ALT-F
Toggle Folding
